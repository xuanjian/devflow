AI agent context starter

Small context first. Private details later.

ai-context-lite gives coding agents a compact, portable entrypoint: read JSON indexes first, install one shared skill, and load detailed project context only when the current task needs it.

Zero dependencies Portable skill links Privacy-first starter 
$ git clone https://github.com/<owner>/ai-context-lite.git
$ cd ai-context-lite
$ npm run doctor
doctor: ok

$ node bin/ai-context-lite.mjs install
Codex: linked ~/.codex/skills/ai-context
Claude: linked ~/.claude/skills/ai-context
Agents: linked ~/.agents/skills/ai-context
Read order

Predictable startup

Agents begin with `config/entry.json`, `runtime/current.json`, and a small project index before reading anything larger.

Install

One skill, many tools

The installer links the same `ai-context` skill into common local agent skill folders without copying files.

Privacy

Generic by default

The starter ships empty. Keep real project maps, task state, and private links in a separate local or private checkout.
