# Reference Archive

This folder keeps long or retired rule/skill/workflow material that should not be loaded by default.

Active catalogs:

- `../../config/rules/rules.json`
- `../../config/skills/skills.json`

Use this folder only when an active rule or skill explicitly points here and the short active entry is not enough.

## Structure

```text
history/          # Migration plans, old roadmaps, generated overviews, and stale architecture notes
rules/            # Retired or long rule material
skills/           # Retired or long skill material
workflows/        # Retired agent/workflow plans
```
