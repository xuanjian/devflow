AI 上下文最小入口

先读小上下文，再加载私有细节。

ai-context-lite 给 AI 编程代理一个紧凑、可迁移的入口： 先读 JSON 索引，安装一个共享 skill，只在当前任务需要时再加载详细项目上下文。

零依赖 可迁移 skill 链接 默认保护隐私 
$ git clone https://github.com/<owner>/ai-context-lite.git
$ cd ai-context-lite
$ npm run doctor
doctor: ok

$ node bin/ai-context-lite.mjs install
Codex: linked ~/.codex/skills/ai-context
Claude: linked ~/.claude/skills/ai-context
Agents: linked ~/.agents/skills/ai-context
读取顺序

启动路径清晰

AI 先读 `config/entry.json`、`runtime/current.json` 和小型项目索引，再决定是否读取更多资料。

安装

一个 skill，多端复用

安装脚本把同一个 `ai-context` skill 链接到常见本地 AI 工具目录，不复制多份文件。

隐私

默认空白通用

公开模板不带真实项目数据。项目地图、任务状态和私有链接放到本地或私有仓库。
